# Python Calculator
A Command Line Based Calculator

## How To Use

The code will prompt you to choose a function. You will need to make sure that they are exactly as how they seem to be on the menu. However, if you make a mistake, the code repeats itself till you get it correct. It will also contain the ability to review and delete your calculators history up to your first run. <br>
After that, the code will ask you for two numbers. You also will be able to repeat the query in the case that your provided input doesn't fit the requirements. <br>

1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Modulus or Remainder
6. View History
7. Delete History
