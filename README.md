# PythonCalculator
A Command Line Based Calculator

## How To Use

The code will prompt you to write 2 numbers. You will need to make sure that they are integers, as that is what the calculator allows. However, if you make a mistake or don't write an integer, the code repeats itself till you get it correct.<br>
After that, the code will ask you what operation you want to do to the two numbers. The options are listed there, but I will list them here as well.<br>

1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Modulus or Remainder
